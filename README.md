# MaxOut Construction Website

Stack: Next.js 14 + Tailwind CSS

## Quick Start (Local)
1. Install Node.js (https://nodejs.org)
2. In this folder run:
   npm install
   npm run dev
3. Open http://localhost:3000

## Deploy on Vercel
- Import this repo at https://vercel.com/new
- Keep defaults (build: `npm run build`, output: `.next`)
- Deploy, then add your domain: maxoutconstruction.ca
